# Datasets

Both CSVs are generated by `sim/notch_sim.py`, which mirrors the on-chain integer math exactly (the same code path the 34-test suite validates against the deployed program).

Reference launch config: start 0.00001 SOL/token, schedule 2x per 25 SOL, buy fee 3% (all to floor), sell fee 6% (1% creator + 5% floor), governor 93.5% minimum backing.

## data/notch-dataset-buys-1to500.csv

One row per sequential 1 SOL buy, from the 1st to the 500th. Columns:

| column | meaning |
|---|---|
| `cum_buys_sol` | cumulative SOL bought (also the row number) |
| `entry_price_sol` | curve price this buyer saw before their buy |
| `tokens_for_1sol` | whole tokens received for 1 SOL |
| `eff_cost_sol` | effective SOL paid per token (1 / tokens) |
| `price_after_sol` | curve price after the buy |
| `nav_sol` | floor per token after the buy (vault / supply) |
| `redeem_price_after_fees_sol` | what a seller receives per token right now (0.94 x NAV) |
| `backing_pct` | NAV / price, the governor keeps this at or above 93.5 |
| `pool_sol` | vault balance |
| `instant_roundtrip_sol` | SOL this buyer would get selling everything immediately |
| `instant_pnl_pct` | the same as a percentage (bounded at about -14.75) |

Checkpoints: buy 1 lands over-backed at 102.2%; from roughly buy 8 the governor pins backing at 93.5% and the whole system follows `NAV = 9.5077e-6 * x^0.09305`.

## data/notch-dataset-sells-after500.csv

Starting from the 500 SOL state, 400 sequential sells of about 1 SOL gross value each. Columns:

| column | meaning |
|---|---|
| `sell_n` | sell number |
| `tokens_sold` | whole tokens redeemed |
| `seller_receives_sol` | payout (constant 0.94 per 1 SOL gross, by construction) |
| `nav_sol` | floor after the sell (strictly rising) |
| `redeem_price_sol` | 0.94 x NAV after the sell |
| `price_frozen_sol` | curve price (never moves on sells) |
| `pool_sol` | vault balance |

What to look for: across 400 consecutive sells the price never moves (sells cannot touch it, only buys advance it), NAV rises every single row (0.00001694 to 0.00001826), and by the end the floor stands above the unchanged price, meaning the next buy prints a new all-time high.
